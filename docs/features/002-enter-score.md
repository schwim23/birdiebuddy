# Feature 002 — Enter Score

## Goal
The user can enter a numeric score for the current hole.

## Requirements
- Round screen shows score buttons (3 through 10)
- Tapping a score saves it for the current hole in memory
- App advances to the next hole
- After hole 18, show a summary screen

## Acceptance Criteria
- Start a round
- Enter 5 on hole 1
- See hole 2 next
- Summary later reflects hole 1 score of 5
