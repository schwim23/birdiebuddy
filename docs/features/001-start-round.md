# Feature 001 — Start Round

## Goal
The user can start a new round and land on hole 1.

## Requirements
- Home screen has a "Start Round" button
- Tapping it creates a new Round with default values
- App navigates to the round screen
- Round screen shows:
  - Hole 1
  - Par 4 (default for MVP)
  - Score not yet entered

## Acceptance Criteria
- User launches app
- Taps Start Round
- Sees Hole 1 screen with par displayed
